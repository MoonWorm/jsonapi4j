# jsonapi4j-oas-plugin

OpenAPI Specification plugin. Automatically generates an OpenAPI-compliant spec from registered resources, relationships, and operations. Exposed as JSON or YAML via a dedicated endpoint.

## Usage

```xml
<dependency>
  <groupId>pro.api4</groupId>
  <artifactId>jsonapi4j-oas-plugin</artifactId>
  <version>${jsonapi4j.version}</version>
</dependency>
```

See [OpenAPI Plugin docs](https://api4.pro/openapi-plugin/) for details.
