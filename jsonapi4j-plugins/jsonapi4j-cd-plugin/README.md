# jsonapi4j-cd-plugin

Compound Documents plugin. Integrates the `jsonapi4j-compound-docs-resolver` into the JsonApi4j plugin pipeline, enabling `include` query parameter support for fetching related resources.

## Usage

```xml
<dependency>
  <groupId>pro.api4</groupId>
  <artifactId>jsonapi4j-cd-plugin</artifactId>
  <version>${jsonapi4j.version}</version>
</dependency>
```

See [Compound Documents docs](https://api4.pro/compound-docs/) for details.
