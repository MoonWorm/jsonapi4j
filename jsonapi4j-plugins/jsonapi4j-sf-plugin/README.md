# jsonapi4j-sf-plugin

Sparse Fieldsets plugin. Supports `fields[TYPE]=field1,field2` query parameter to return only requested attributes per resource type, reducing payload size.

## Usage

```xml
<dependency>
  <groupId>pro.api4</groupId>
  <artifactId>jsonapi4j-sf-plugin</artifactId>
  <version>${jsonapi4j.version}</version>
</dependency>
```

See [Sparse Fieldsets docs](https://api4.pro/sparse-fieldsets/) for details.
