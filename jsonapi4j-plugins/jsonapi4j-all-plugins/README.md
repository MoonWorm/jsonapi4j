# jsonapi4j-all-plugins

Convenience aggregator module that includes all default JsonApi4j plugins as a single dependency.

## Includes

- `jsonapi4j-ac-plugin` — Access Control
- `jsonapi4j-cd-plugin` — Compound Documents
- `jsonapi4j-sf-plugin` — Sparse Fieldsets
- `jsonapi4j-oas-plugin` — OpenAPI Specification

## Usage

```xml
<dependency>
  <groupId>pro.api4</groupId>
  <artifactId>jsonapi4j-all-plugins</artifactId>
  <version>${jsonapi4j.version}</version>
</dependency>
```
